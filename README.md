# Text-Based-Sentiment-Analyzer-with-Gradio
Developed a custom AI model to analyze text based input and classify it into 1 of 3 classes (Positive, Negative, Neutral). The model achieved an accuracy of 87.6%. This was a joint effort of me and 2 of my friends.
In order to run this project, you can use Google Colab or Jupyter Notebook to run the project.ipynb file. The model file would be available after extracting the sentiment_model.rar compressed file into the project directory.
